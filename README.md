# on-china-guide

本仓库用于归档和发布一组独立的 HTML 页面，内容涵盖与中国相关的信息整理、指南与参考资源。所有页面均为静态 HTML 文件，不依赖后端服务，可直接在浏览器中打开浏览。

## 项目简介

`on-china-guide` 是一个以归档为目的的仓库，存放多个主题独立、格式统一的信息页面。每一页均可单独访问，适合需要离线或在线快速查阅的场景。项目本身不绑定任何特定域名或平台，仅作为页面集合的托管与版本管理。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 首页/导航页（如存在）
└── pages/             # 存放所有独立 HTML 页面
    ├── example-1.html
    ├── example-2.html
    └── ...
```

- `index.html`：可选，作为仓库的入口或页面索引。
- `pages/`：主要页面存放目录，每个 HTML 文件代表一个独立页面。

## 页面归档说明

- 所有页面均为纯 HTML 格式，无外部依赖（如数据库、服务器脚本）。
- 页面内容通常以信息整理、参考或指南为主，不包含商业推广或营销信息。
- 每个页面可独立使用，也可通过导航页统一访问。
- 页面内容可能随时间更新，历史版本可通过 Git 提交记录追溯。

## 维护说明

本仓库由维护者根据实际需求不定期更新。更新内容可能包括：

- 新增页面
- 修正已有页面中的错误或过时信息
- 调整页面结构或样式

欢迎通过 Issues 提交反馈或建议。对于页面内容的修改，建议先通过 Issues 讨论，再提交 Pull Request。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/on-china-guide.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 若仓库包含 `index.html`，也可通过该文件访问所有页面索引。
